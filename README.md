# 🚀 Anish AI Scheduler

An AI-powered intelligent scheduling system that automatically generates optimized timetables using constraint-based logic and smart decision-making.

## 📌 Overview
Anish AI Scheduler is designed to simplify timetable creation by automating the scheduling process. It intelligently assigns subjects, time slots, and resources while ensuring balance and avoiding conflicts. This system reduces manual effort and improves planning efficiency.

## ✨ Key Features

### 🧠 AI-Based Timetable Generation
Generates schedules automatically based on user inputs such as subjects, time slots, and constraints.

### ⚙️ Constraint-Based Scheduling
Applies rules like:
- No repetition of the same subject continuously  
- Balanced subject distribution  
- Proper time allocation  

### 🔄 Conflict Resolution
Ensures there are no overlapping schedules or resource conflicts.

### 📊 Optimized Scheduling
Creates efficient and well-balanced timetables to improve productivity.

### 💻 User-Friendly Interface
Simple and responsive UI for easy interaction and better usability.
